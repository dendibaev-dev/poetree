## Poetree
Это система, которая позволяет пользователям отправлять текстовые заметки размером до 280 символов с помощью веб-интерфейса

## Stack of languages/technologies:
- Javascript(es6+)
- **ReactJs**
- react-router-dom
- **redux**
- redux-toolkit
- normalizecss
- react-hook-form
- react-icons
- **Firebase Auth**
- **Firestore**
